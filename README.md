# CampusKona 🛒 
### The Official Student Marketplace 

CampusKona is a lightweight, fast, and mobile-friendly web application designed to help students at the Kwame Nkrumah University of Science and Technology (KNUST) buy and sell items within the campus community.


## 🚀 Live Demo
[View the Live App Here](https://github.com/lilspek/campuskona.git)


## ✨ Features
* **Instant Search:** Find hostel essentials, books, or electronics in seconds.
* **Category Filtering:** Browse specific categories like Tech, Books, or Hostel supplies.
* **WhatsApp Integration:** Connect buyers and sellers directly via a pre-filled WhatsApp message.
* **Image Support:** High-quality image previews powered by the ImgBB API.
* **Admin Controls:** Simple "Mark as Sold" feature to keep the marketplace fresh.

## 🛠️ Technical Stack
* **Frontend:** HTML5, CSS3 (Custom Grid System), JavaScript (ES6+)
* **Database:** Google Firebase Firestore (Real-time NoSQL)
* **Image Hosting:** ImgBB API
* **Deployment:** GitHub Pages

## 🛡️ Security & Privacy
* **Domain Whitelisting:** Firebase is configured to only accept requests from authorized domains.
* **Protected Deletion:** Item removal is protected by an admin-defined password to prevent unauthorized tampering.
